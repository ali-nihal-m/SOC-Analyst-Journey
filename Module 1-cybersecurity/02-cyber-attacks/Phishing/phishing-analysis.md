# Phishing Attack Analysis

## Definition

Phishing is a social engineering attack used to steal credentials or sensitive information.

## Attack Flow

Attacker
↓
Fake Email
↓
Victim Clicks Link
↓
Fake Website
↓
Credentials Stolen

## Indicators

- Suspicious sender
- Misspelled domains
- Urgent language
- Unexpected attachments

## Prevention

- MFA (MUlti Factor Authentication)
- User awareness
- Email filtering

## SOC Detection

- Email alerts
- User reports
- Suspicious login activity
